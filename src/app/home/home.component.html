About Me

The main goal was to create a modern and minimalistic portfolio website for a blockchain app developer. After some research, I came up with the idea of how to create the website using the modern approach. Using some basic colors and dark theme I was able to create a modern and unique look. The website replicates visual studio code. This makes the viewer feel more comfortable while exploring the website and also helps with the eye comfort considering the increasing use of dark mode.
